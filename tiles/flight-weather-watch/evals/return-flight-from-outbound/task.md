# Return Flight Briefing

## Problem/Feature Description

A pilot is planning a day trip. They already have a briefing for the outbound leg: San Carlos to Bend Municipal (KBDN), departing Monday March 30, 2026 at 8:00 AM Pacific Daylight Time, 12,000 feet, 160 knots TAS.

Now they need a briefing for the return flight: Bend Municipal back to San Carlos, departing the same day at 2:30 PM local time (Pacific Daylight Time), same altitude and airspeed.

Generate the shell command for the return leg briefing.

## Output Specification

Produce the exact `python3 flightweather.py` command for the return flight. Write it to `run_briefing.sh`.
